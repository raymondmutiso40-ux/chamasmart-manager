# chamasmart-manager
chamasmart manager its a chama management sytem that is designed to improve transparency ,tracking and financial predictions
## Features
 Add and manage members
 track contributions in real-time
 automatic total contribution
 loan eligibility detection (rule based)
 predictive growth estimation (basic ai logic)
 simple and mobile-friendly features
## Tech Stack
  Frontend:HTML,CSS,JavaScript
  Backend:PHP(XAMPP)
  Database:MySQL
  Logic:simulated AI
